POLAR CHART UI

This one was super fun. I messed around with bar charts and line charts first before settling on building a simple Polar Chart UI using the base chart + legend initiation and "grab data" methods provided by the Chart.js API.

Stack: Chart.js, jQuery, Javascript, HTML/LESS/CSS
