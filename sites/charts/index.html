Project: Graphic UI built on base Polar Chart functiontionality provided by Chart.js api.

Chart.js provides base functions to intialize(draw) the chart and its accompanying legend on page load.

It also provides an extremely basic function to grab the data of a slice on click event and update the chart off present data.


I implemented a "Loop search" deleting and a custom updating function hinging on the slice's "label" value to allow users to delete and update both the data and the resulting chart, as well as created a UI that strings and glues the functions together for a seamless polar chart maker.

Definitely had a lot of fun creating the random color generator using Math.Random() and an array for hex values.

Current Stack: HTML/Less/lots of jQuery
