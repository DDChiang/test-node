POLAR CHART UI

This one was uber fun. I messed around with bar charts and line charts first before settling on building a simple Polar Chart UI using the base chart + legend initiation and "grab data" methods provided by the Chart.js API.

What I added: Random hex color generator, a way for users to dynamically delete or update data slices, and a simple UI for users to build Polar Charts off their own data.

Stack: Chart.js, jQuery, Javascript, HTML/LESS/CSS

Tags: Dynamic Chart Rendering, JSON, Data, UI
